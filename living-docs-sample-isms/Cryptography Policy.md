## Purpose

This policy defines the cryptographic controls used to protect organisational data.

## Data in transit

Data transmitted over untrusted networks is encrypted. The organisation supports current
transport encryption protocols.

## Data at rest

Sensitive data at rest is encrypted using industry-standard algorithms. Keys are managed in the
organisation's key-management service.

## Key management

Keys are rotated on a defined schedule and access to key material is restricted and logged.
