## Purpose

This statement records which security controls apply to the organisation, whether each is
implemented, and the policy that governs it.

## Access control

Access control is implemented and governed by the Access Control Policy.

## Cryptography

Cryptographic controls are implemented and governed by the Cryptography Policy.

## Logging and monitoring

Logging and monitoring are implemented and governed by the Logging and Monitoring Policy.

## Supplier relationships

Supplier security is implemented and governed by the Supplier Security Policy.
