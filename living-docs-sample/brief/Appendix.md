# Appendix — Design Tokens

This appendix lists the concrete values referenced by the Project Brief and the
Executive Summary. Implementation should read these tokens rather than hard-coding
values per surface.

## Colour tokens

- The primary colour is blue (#1E5BFF) — used for the primary button, active links,
  and focus rings.
- The neutral ink is #14161A for body text.
- The surface background is #FFFFFF.

## Usage notes

Only one primary-colour element should compete for attention in a given view. When in
doubt, demote a blue element to neutral so the principal action stays unmistakable.
